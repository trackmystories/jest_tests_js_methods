# TESTS FOR JAVASCRIPT METHODS WITH JEST

##### set up

```
 npm install

```

##### run tests

```

 yarn test

 or

 npm run test

```

Test the following JS methods:
• Array.prototype.map();
• Array.prototype.push();
• Array.prototype.forEach();
• Function.prototype.call()
• Function.prototype.apply()
• Object.prototype.toString()
